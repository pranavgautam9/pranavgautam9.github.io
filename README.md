**This Portfolio** is a personal website built to showcase my work as a software engineer. The site features a clean, modern design with smooth animations, responsive layout, and an intuitive user experience. It highlights my experience in full-stack development, mobile app development, and various projects I've built.

The portfolio includes sections for About, Experience, Projects, and Contact, providing visitors with a comprehensive view of my professional journey and technical capabilities.

## üåü Features

### Core Functionality

#### 1. Responsive Design
- **Mobile-First Approach**: Fully responsive layout that works seamlessly across all devices
- **Adaptive Navigation**: Horizontal sidebar on desktop, bottom bar on mobile
- **Flexible Grid Layouts**: Projects and content adapt to different screen sizes
- **Touch-Friendly Interface**: Optimized for mobile interactions

#### 2. Interactive Sections
- **About Me**: Personal introduction with education and background
- **Experience**: Tabbed interface showcasing work history with detailed descriptions
- **Projects**: Featured project display with grid of additional projects
- **Contact**: Easy-to-use contact section with social links

#### 3. Visual Enhancements
- **Smooth Animations**: Scroll-triggered animations and transitions
- **Grayscale Image Effects**: Project images start in grayscale and transition to color on hover
- **Active Navigation**: Highlights current section in navigation bar
- **Loading Animation**: Elegant loader on page initialization

#### 4. User Experience
- **Smooth Scrolling**: Seamless navigation between sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Social Links**: Quick access to GitHub, LinkedIn, LeetCode, and email
- **Resume Download**: Direct link to downloadable resume
